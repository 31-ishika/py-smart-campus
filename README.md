# EduFlow — Smart Campus System

A Next.js demo app named `smart-campus-system` — EduFlow academic management dashboard.

Features:
- Dark-themed dashboard with cards for Fee Calculator, File Handling, Directory Scanner, Performance Analytics
- Sidebar navigation for various sections
- Analytics page with student table and Pie chart (Chart.js)
- CSV export for analytics

Quick start:

```bash
cd smart-campus-system
npm install
npm run dev
```

Open http://localhost:3000

Tech: Next.js, React, Tailwind CSS, Chart.js
